## Breve Descrição da Proposta

O FoodBridge CLI é uma aplicação de linha de comando em Python que visa conectar doadores de alimentos (restaurantes, mercados) a entidades receptoras (ONGs, abrigos) para combater o desperdício de alimentos e a insegurança alimentar. A aplicação permite o registro, listagem e coleta de doações, utilizando um arquivo JSON para persistência de dados. O projeto foi desenvolvido seguindo boas práticas de engenharia de software, incluindo versionamento semântico, testes automatizados, linting e integração contínua via GitHub Actions.
